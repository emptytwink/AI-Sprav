README для тестового архива
===========================

Это демонстрационный набор документов для проверки функции «ИИ-сбор справочника».
Архив не содержит реальных корпоративных данных. Документы описывают условный комплекс КСД-01.
Ожидаемый результат работы ИИ: разделы «Общие сведения», «Архитектура», «Монтаж», «Эксплуатация», «Техническое обслуживание», «Диагностика», «Безопасность», «Обновление ПО», «Чертежи и схемы», «Комплектность».
